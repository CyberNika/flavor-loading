# fu-loading

### 概述
Loading Component based on React.

基于 React 的 Loading 组件。

# 目录结构
```
fu-loading/
├── README.md
├── dist          # 构建目录
├── docs          # 文档
├── examples      # 示例
├── gulpfile.js   # 构建配置文件
├── LICENSE
├── package.json  # NPM 依赖包文件
└── src           # 源文件
```

# 主题和示例

### 主题
- circle
  - &
  - &-fadeout
  - &-zoomfade
- ball
  - &-bounce
- rect
  - &
  - &-series

### 属性
- transparent  `boolen`
  - `true`  透明/白色
  - `false`  非透明/实色
